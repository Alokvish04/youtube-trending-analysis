# YouTube Trending Videos Analysis

This project analyzes YouTube trending videos using Python, pandas, and visualization libraries. It explores patterns in views, likes, comments, title length, and other features using the public Kaggle Trending YouTube Video Statistics dataset. [web:1]

## Project Structure

- `youtube_trending.ipynb` – main Jupyter notebook with data cleaning, EDA, and visualizations.
- `archive/` – dataset files (country-wise CSVs such as `USvideos.csv`, `INvideos.csv`, etc.). [web:1]

## Tech Stack

- Python
- pandas, numpy
- matplotlib, seaborn
- wordcloud

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Alokvish04/youtube-trending-analysis.git
   cd youtube-trending-analysis
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start Jupyter Notebook and open:
   ```bash
   jupyter notebook youtube_trending.ipynb
   ```

You can then run the notebook cells to reproduce the analysis and plots.
